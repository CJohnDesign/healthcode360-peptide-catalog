# HealthCode360 Peptide Catalog — PRD

Version 2.2 | Effective Date: January 31, 2026

## Overview

A sophisticated, Porsche Design-inspired peptide catalog for HealthCode360. Stark black/white/grey aesthetic, full-bleed placeholder imagery, 26 hardcoded products across 6 categories, filterable catalog, interest list flow, and knowledge base. No auth, no checkout.

## Product Catalog

See plan for full 26-product list. Categories: Weight Management, Growth Hormone & Recomposition, Healing & Tissue Recovery, Reproductive & Hormonal Health, Longevity & Anti-Aging, Wellness & Mood.

## Key Flows

- **Browse**: Home → Categories / Products → Product Detail
- **Search**: Navbar search → Search results (products, articles, glossary)
- **Interest List**: Add products → Review → Submit Inquiry (mock)
- **Compare**: Up to 3 products side-by-side

## Tech Stack

Next.js 14 App Router, Tailwind CSS, React Context (Interest List), hardcoded data.
